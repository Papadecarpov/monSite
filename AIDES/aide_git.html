Aide sur Git

Définir son nom et email :
git config --global user.name "Votre nom ou pseudo"
git config --global user.email "votre@email.com"

Pour activer un dossier comme repository Git, se placer dans ce dossier puis :
git init

Pour ajouter à l'index Git un fichier nouvellement créé :
git add checklist-vacances.md

Pour ajouter tous les fichiers du répertoire courant :
git add .

Après une modification du projet :
git commit -m "message pour commenter le commit"

Pour mettre à jour tous les fichers déjà présents dans l'index :
git commit -a -m "message pour commenter le commit"

Pour afficher l'historique des commit :
git log

Pour se positionner sur un commit :
git checkout SHADuCommit

Pour se positionner sur le dernier commit :
git checkout master

Aide sur GitHub

Pour récupérer un repository sur sa machine, ce positionner sur le répertoire de réception et :
git clone lienhttpsfournipargithub

Pour créer un repository sur gitHub, cliquer sur le bouton "Create new" ou "+" du site
Pour synchroniser le repository local et celui de GitHub, se placer sur le répertoire local et :
git push origin master

Inversement, pour récupérer les modifications effectuées sur GitHub sur sa machine locale :
git pull origin master
