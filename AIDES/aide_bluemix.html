Aide sur Bluemix :

Se positionner sur son répertoire de travail :
cd votre_nouveau_répertoire

Se connecter à IBM CLOUD :
bluemix api https://api.ng.bluemix.net
bluemix login -u yannick.colin@tessi.fr -o nom_org -s nom_espace

Dans le cas d'un ID fédéré :
bluemix login -o nom_org -s nom_espace -sso

Pour redéployer son application dans IBM CLOUD :
bluemix app push mon_application
